# **App Name**: EZCirkit E-Commerce

## Core Features:

- Product Catalog: Display and manage electronic learning kits with detailed product information.
- Shopping Cart System: Enable users to add, update, and remove items from their cart, with real-time updates and cart persistence using Firestore.
- User Authentication: Implement secure user authentication using Firebase Auth with email/password, Google Sign-In, and phone number OTP options.
- Checkout Process: Guide users through a multi-step checkout process with form validation and address management.
- Payment Integration: Integrate Razorpay for secure payment processing with various payment options.
- Order Management: Allow customers to view their order history and track order status. Provide an admin dashboard for managing and updating orders.
- Wishlist Feature: Enable users to add products to a wishlist and manage saved items.

## Style Guidelines:

- Primary gradient: Linear gradient from orange (#FF6B35) to light-orange (#FF8A65) for a vibrant and engaging feel.
- Background: White (#FFFFFF) with subtle gradient sections to maintain a clean and modern look.
- Accent: Use orange (#FF6B35), green (#4CAF50), and gold stars (#FFB800) for key elements and interactive components.
- Font: 'Segoe UI', a sans-serif font for clean and modern readability.
- Use Font Awesome icons for visual cues and interactive elements.
- Implement a responsive layout with breakpoints at 768px (tablet) and 480px (mobile) to ensure a seamless experience across devices.
- Incorporate smooth animations and hover effects with 0.3s transitions for an interactive and engaging user experience.
